## 🧭 Mapa de 30 segundos

```
📊 Vendas       — KPIs, funil, ranking, bônus
📣 Marketing    — Tráfego e CAC
💬 CS           — Alunos, tickets, NPS, renovações
🏢 Workspace    — Sua home (Meu Dia) + tarefas + materiais
🛡️ Administração — Cadastros (só admin)
⚙️ Configurações — Perfil, segurança, dados
```

> 📱 **Em mobile:** toque no **☰** no topo pra abrir o menu

## ✅ Checklist de teste rápido

### 1. Login (1 min)
- [ ] Entrar com email/senha
- [ ] Cai em Dashboard Times ✓

### 2. Cadastros básicos (5 min) — *admin only*
- [ ] **Administração → Vendedores**: criar 3 vendedores
- [ ] **Administração → Cursos**: criar 2 cursos (R$ 997 e R$ 2.997)
- [ ] **Administração → Metas Mensais**: definir meta de R$ 50.000 para cada vendedor
- [ ] **Administração → Registrar Vendas**: registrar 5 vendas

### 3. Validação Vendas (2 min)
- [ ] **Vendas → Dashboard Times**: KPIs com números, gráfico aparece
- [ ] **Vendas → Ranking**: pódio top 3 aparece

### 4. CS (4 min)
- [ ] **CS → Alunos**: cadastrar 2 alunos
- [ ] Abrir um aluno → aba **Matrículas** → adicionar matrícula
- [ ] Mesma tela → aba **NPS** → registrar nota 9 com comentário
- [ ] Mesma tela → aba **Onboarding** → marcar 2-3 etapas
- [ ] **CS → Dashboard** → KPIs refletem

### 5. Workspace (2 min)
- [ ] **Workspace → Tarefas**: criar 2 tarefas (uma com prazo de ontem pra ver "atrasada")
- [ ] **Workspace → Materiais**: criar 1 script com tags
- [ ] **Workspace → Meu Dia**: tarefa e material aparecem

### 6. Mobile (1 min)
- [ ] Abrir no celular OU DevTools modo mobile
- [ ] Menu ☰ funciona como drawer
- [ ] Cards adaptam (1 coluna no mobile)

5. F12 → Console → screenshot dos erros vermelhos

## ❓ Dúvidas comuns

**"Está lento ou em loading infinito"**
→ Hard refresh (Ctrl+Shift+R) ou aba anônima

**"Cadastrei mas não apareceu"**
→ F5 na página (cache de 5 min)

**"Não consigo criar venda — botão desabilitado"**
→ Falta vendedor OU curso cadastrado

**"Não vejo Administração no menu"**
→ Sua conta não é admin. Peça ao gestor pra promover
